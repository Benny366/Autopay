📖 Quick Setup Guide
📧 Gmail Setup
1. Go to Google Account → Security
2. Enable 2-Step Verification
3. Search "App Passwords" → Generate
4. Select "Mail" → Copy 16-char password
5. Paste above as Gmail App Password
6. Use your Gmail as SMTP_USER
📱 Twilio SMS Setup
1. Sign up at twilio.com (free trial)
2. Get Account SID + Auth Token from Console
3. Get a free Twilio phone number
4. Add your phone number as verified caller
5. Paste credentials above
6. Set SMS_ENABLED = True in app.py
